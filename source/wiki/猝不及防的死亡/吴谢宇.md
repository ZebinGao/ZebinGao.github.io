# 吴谢宇执行死刑事件汇总

> **摘要**：2024年1月31日，“北大学子弑母案”罪犯吴谢宇被依法执行死刑。这起跨度长达9年的案件，因其当事人的“学霸”身份、作案手段的残忍冷静以及长达3年的潜逃生涯，引发了全社会关于家庭教育、心理健康及高智商犯罪的持久讨论。

## 1. 案件终局：执行死刑

### 执行详情
* **执行时间**：2024年1月31日
* **执行机关**：福州市中级人民法院（遵照最高人民法院下达的执行死刑命令）
* **临刑安排**：
    * 执行前，法院依法告知吴谢宇可以申请会见亲属。
    * **特殊细节**：吴谢宇**未提出申请**会见亲属。（此前报道显示，其父系亲属曾表示希望见最后一面，但最终未能成行，原因在于吴谢宇本人的放弃）。
* **法律程序**：最高人民法院复核认为，吴谢宇所犯罪行极其严重，主观恶性极深，虽然到案后如实供述，但不足以从轻处罚，故核准死刑。

## 2. 人物与背景
* **姓名**：吴谢宇
* **生卒年**：1994年10月7日 - 2024年1月31日（终年29岁）
* **身份标签**：
    * 前北京大学经济学院学生
    * 曾经的“宇神”、“完美小孩”
    * 弑母案凶手
* **家庭背景**：父亲吴志坚（2010年因病去世），母亲谢天琴（中学历史教师，被害人）。

## 3. 案件时间线回顾

| 时间节点 | 事件详情 |
| :--- | :--- |
| **2010年** | 吴谢宇父亲因病去世，据其供述，此后他开始产生悲观厌世情绪。 |
| **2015年7月10日** | 吴谢宇在福州家中，趁母亲谢天琴换鞋之际，持哑铃杠连续猛击其头面部致死。 |
| **2015年7月-8月** | 吴谢宇通过网购材料（活性炭、塑料膜等）将尸体层层包裹（75层）并放入冰柜，安装监控与报警器。 |
| **2015年下半年** | 以“陪母亲出国交流”为由，向亲戚骗取借款144万元，并挥霍于享乐。 |
| **2016年2月14日** | 尸体在福州某中学教职工宿舍内被发现。警方锁定吴谢宇为嫌疑人并发布悬赏。 |
| **2016年-2019年** | 吴谢宇潜逃三年，期间购买30多张身份证，曾在重庆等地以“男模”身份在酒吧工作。 |
| **2019年4月21日** | 吴谢宇在重庆江北国际机场送机时被警方抓获。 |
| **2021年8月26日** | **一审宣判**：福州中院以故意杀人罪、诈骗罪、买卖身份证件罪数罪并罚，判处吴谢宇死刑。吴不服上诉。 |
| **2023年5月30日** | **二审宣判**：福建高院裁定驳回上诉，维持原判，并报请最高法核准死刑。 |
| **2024年1月31日** | **执行死刑**。 |

## 4. 核心争议与关注点

### 作案动机
* **吴谢宇自述**：声称为了帮母亲“解脱”，认为父亲死后母亲生活痛苦，自己想在杀母后自杀（但最终未实施自杀）。
* **法院认定**：法院认为其悲观厌世产生杀念，但其作案后精心清理现场、诈骗巨款并挥霍、隐匿身份潜逃等行为，说明其具有极强的求生欲和逻辑反侦察能力，驳回了其“帮助自杀”或“精神病”辩护。

### 双面人生
* **A面**：他是亲戚邻居眼中的乖孩子，老师同学眼中的天才学霸，GRE成绩排名前5%的北大学生。
* **B面**：他是长期压抑自我、涉足色情场所、购买性服务、甚至在逃亡期间做“男公关”的纵欲者。

## 5. 社会反思
吴谢宇案不仅仅是一起刑事案件，更成为了中国社会心理学和家庭教育的典型反面教材：
* **“空心病”现象**：引发了关于高学历人才心理健康与价值观缺失的讨论。
* **亲子关系异化**：反思过度紧密、高期待且缺乏情感流动的“共生”亲子关系如何导致悲剧。
* **完美主义的代价**：他在庭审中曾提到“从小到大就像在演戏”，为了满足母亲和外界的期待而活，最终导致人格崩塌。
