# 复旦大学教师于娟逝世事件汇总

> **摘要**：2011年4月19日，复旦大学社会发展与公共政策学院青年教师、海归博士于娟，在与晚期乳腺癌抗争一年四个月后不幸辞世，年仅32岁。她在病榻上撰写的《癌症日记》及其遗作《此生未完成》，以直面生死的坦诚和深刻的生命反思，震撼了无数正在透支健康的年轻人，成为关于生命教育的经典读本。

## 1. 逝者档案
* **姓名**：于娟
* **生卒年**：1978年4月2日 - 2011年4月19日
* **享年**：32岁
* **籍贯**：山东济宁
* **身份**：
    * 复旦大学社会发展与公共政策学院 讲师
    * 挪威奥斯陸大學 经济学硕士
    * 复旦大学 经济学博士
    * 图书《此生未完成》作者
* **家庭**：丈夫（网名“光头”）、儿子（小名“土豆”，当时仅2岁多）、父母。

## 2. 抗癌与逝世时间线

| 时间节点 | 事件详情 |
| :--- | :--- |
| **2009年12月** | 于娟突发腰痛，后被确诊为**乳腺癌晚期**（IV期），且癌细胞已扩散至骨骼，错过了手术机会。 |
| **2010年1月** | 确诊后，她在病榻上开设博客“活着就是王道”，开始记录抗癌经历及对过往生活方式的深度反思。 |
| **2010年-2011年** | 经历多次化疗、放疗及剧烈疼痛的折磨，坚持写下70余篇“癌症日记”，引发网络巨大关注。 |
| **2011年4月2日** | 在病房度过了自己最后的33岁（虚岁）生日，此时身体状况已极度恶化。 |
| **2011年4月19日** | 凌晨3时许，于娟在上海辞世。 |
| **2011年6月** | 其遗作《此生未完成》正式出版。 |

## 3. 核心思想与《此生未完成》
于娟的文字之所以产生巨大影响力，不在于煽情，而在于她以生命为代价，对现代人病态生活方式进行了“血淋淋”的剖析。

### 3.1 经典语录
> * “在生死临界点的时候，你会发现，任何的加班，给自己太多的压力，买房买车的需求，这些都是浮云。如果有时间，好好陪陪你的孩子，把买车的钱给父母亲买双鞋子，不要拼命去换什么大房子，和相爱的人在一起，蜗居也温暖。”
> * “长期熬夜等于慢性自杀。”
> * “生不如死九死一生死里逃生死死生生之后，我突然觉得，一生轻松。不想去控制大局小局，不想去多管闲事淡事，我不再有对手，不再有敌人，我也不再关心谁比谁强，课题录用、职称晋升……”

### 3.2 深度反思
她在博文中详细复盘了自己患癌前的“作死”行为，总结为以下几点，警醒世人：
1.  **瞎吃八吃**：暴饮暴食，嗜荤如命，曾因贪吃野味而不仅不悔，反而引以为豪。
2.  **疯狂熬夜**：长期为了考试、考研、写论文而通宵，自称“2W女”（晚间11点甚至12点前不睡觉）。
3.  **高压性格**：争强好胜，对自己要求极高，为了学业和工作拼命，长期处于高压状态。
4.  **环境问题**：提及家具甲醛超标等环境因素可能的影响。

## 4. 社会影响与遗产
* **“于娟效应”**：她的离世引发了当时职场人、高校师生对“拿命换钱/换学历”价值观的集体反思，推动了公众对健康管理的重视。
* **能源林**：遵照其遗愿，她的骨灰被带回山东老家，安葬于一片公益“能源林”中，寄托了她对环保事业的关注。
* **生命教材**：《此生未完成》一书至今仍是许多人推荐的生命教育书籍，提醒人们在忙碌的追逐中停下脚步，思考“活着的意义”。
