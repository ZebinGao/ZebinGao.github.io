# 符月华（Fu Yuehua）逝世及案件始末汇总

> **摘要**：2025年1月17日，曾以全县理科状元身份考入华东师范大学的 22 岁女生符月华，因长期遭受高中班主任唐毓文的性侵与精神控制，在重度抑郁的折磨下烧炭自杀身亡。在其去世后，家属通过整理遗物发现真相并实名举报，最终促使施暴者受到法律制裁。

## 1. 逝者档案
* **姓名**：符月华
* **生卒年**：2002年6月1日 - 2025年1月17日
* **享年**：22岁
* **籍贯**：广西百色市那坡县
* **身份**：
    * 原广西百色祈福高中 2017 级学生
    * 原华东师范大学物理与电子科学学院学生（后因病退学）
* **死因**：一氧化碳中毒（烧炭自杀）

## 2. 悲剧背景与成因
符月华的悲剧被舆论称为现实版《房思琪的初恋爱乐园》。
* **施暴者**：唐毓文（原百色祈福高中物理教师、班主任，曾获“模范教师”称号）。
* **侵害事实**：据法院审理查明，2019年9月至11月期间（实际跨度可能始于2017年高一时期），唐毓文利用班主任的职务便利，对患有抑郁症的未成年学生符月华实施了精神控制、强制猥亵及强奸。
* **精神创伤**：长期的侵害导致符月华患上严重的创伤后应激障碍（PTSD）和重度抑郁症，出现自残、幻听及多次自杀未遂行为，最终导致其无法完成学业并走向绝路。

## 3. 时间线回顾

| 时间节点 | 事件详情 |
| :--- | :--- |
| **2017年** | 符月华以全县第一名的成绩考入百色祈福高中。分班后，唐毓文成为其班主任，随后开始以“补课”、“关爱”为名对其进行接触与侵害。 |
| **2019年** | 符月华被确诊为重度抑郁症，并在日记中记录了被性侵的痛苦细节，提到“像房思琪一样”。 |
| **2020年7月** | 因精神崩溃，符月华在高考考场上剪碎准考证，中途弃考。 |
| **2021年** | 复读一年后，她以615.5分（全县理科状元）的成绩考入华东师范大学。 |
| **2022年** | 入学仅半年后，因抑郁症复发无法正常生活，被迫休学，后退学回家。 |
| **2025年1月17日** | 符月华在广西某地出租屋内烧炭自杀，不幸离世。 |
| **2025年2月10日** | 符月华表姐农淳鈺整理遗物时发现日记与聊天记录，发布实名举报视频，引发全网关注。 |
| **2025年2月11日** | 百色市教育局通报唐毓文被停职，公安机关立案侦查。 |
| **2025年12月15日** | **一审宣判**：广西百色市右江区人民法院判决唐毓文犯强奸罪、强制猥亵罪，数罪并罚，**执行有期徒刑十八年**，剥夺政治权利五年，并终身禁止从事密切接触未成年人的工作。 |

## 4. 关键证据与遗言
家属在符月华的手机、电脑及日记本中发现了大量证据，这些内容成为了定罪的关键：
* **日记控诉**：详细记录了唐毓文如何在办公室内对其进行猥亵和强奸，以及施暴后的精神洗脑（PUA）言论，如“师母和你我都爱”。
* **文言文检讨书**：遗物中发现一篇被唐毓文要求写下的检讨书，其中隐晦地使用了“腹痛”等词汇暗示身体受到的侵害。
* **生前遗言**：她在社交媒体小号及日记中多次流露出对人生的绝望，称自己“已经烂掉了”，“灵魂被杀死了”。

## 5. 司法正义与社会影响
* **重判信号**：法院在判决中特别指出，唐毓文的行为虽未直接导致被害人当场死亡，但对被害人抑郁症加重并最终自杀有重大影响，且利用教师职业便利犯罪，性质极其恶劣，故依法**从重处罚**（18年刑期接近有期徒刑的顶格）。
* **舆论反响**：
    * 该案件引发了社会对“师德师风”建设的强烈关注，公众呼吁建立更完善的校园性骚扰防范机制。
    * “房思琪式”的沉默螺旋再次被讨论——受害者往往因为羞耻感、权力不对等而长年不敢发声，最终酿成不可挽回的后果。
